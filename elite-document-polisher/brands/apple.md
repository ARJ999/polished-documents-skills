# Apple Brand Guide

## Overview
Apple is the world's most valuable company, known for revolutionary consumer products. Their visual identity embodies premium minimalism, elegant simplicity, and refined sophistication.

## Visual Identity

### Typography
- **Headings**: Arial (sans-serif) - Medium weight, often NOT bold
- **Body**: Arial (sans-serif) - Regular weight
- **Characteristic**: Light weight headings, generous spacing

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Apple Blue | #0071E3 | Links, CTAs |
| Apple Link Blue | #0066CC | Secondary accents |
| Near Black | #1D1D1F | Primary text |
| System Gray | #86868B | Captions, secondary text |
| White | #FFFFFF | Background |

### Design Principles
- Extreme minimalism
- Generous whitespace
- Large border-radius (12px)
- Light touch typography
- Premium simplicity

## Best Used For
- Product documentation
- User guides
- Support articles
- Consumer manuals
- Getting started guides
- Feature explanations

## Style Specifications
```
H1: Arial 32pt, #1D1D1F (Near Black), Regular (NOT Bold)
H2: Arial 24pt, #1D1D1F, Regular (NOT Bold)
H3: Arial 16pt, #1D1D1F, Bold
Body: Arial 12pt, #1D1D1F
Caption: Arial 10pt, #86868B
```

## Document Characteristics
- Minimal clean table style
- Links-only accent usage
- Premium, understated elegance
- Maximum breathing room
- Consumer-friendly clarity
