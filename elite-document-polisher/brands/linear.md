# Linear Brand Guide

## Overview
Linear is the modern issue tracking tool built for high-performance software teams. Their visual identity represents cutting-edge tech aesthetics with distinctive purple tones.

## Visual Identity

### Typography
- **Headings**: Arial (sans-serif) - Medium weight
- **Body**: Arial (sans-serif) - Regular weight
- **Characteristic**: Modern, clean, precise

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Linear Purple | #5E6AD2 | Primary accent, badges |
| Linear Gray | #A1A7C1 | Secondary accents |
| Near Black | #1A1A1A | Primary text |
| Mid Gray | #6B6B6B | Secondary text, captions |
| White | #FFFFFF | Background |

### Design Principles
- Ultra-modern aesthetic
- Moderate border-radius (8px)
- Precise, systematic layouts
- Purple as signature element
- Engineering precision

## Best Used For
- Product specifications
- Changelogs
- Release notes
- Engineering documentation
- Sprint documentation
- Technical roadmaps

## Style Specifications
```
H1: Arial 32pt, #1A1A1A (Near Black), Bold
H2: Arial 24pt, #1A1A1A, Bold
H3: Arial 16pt, #1A1A1A, Bold
Body: Arial 11pt, #1A1A1A
Caption: Arial 9pt, #6B6B6B
```

## Document Characteristics
- Modern subtle table style
- Badges and status accent usage
- Contemporary tech aesthetic
- Clean, focused layouts
- Product team-oriented
