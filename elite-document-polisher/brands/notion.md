# Notion Brand Guide

## Overview
Notion is the all-in-one workspace for notes, docs, wikis, and project management. Their visual identity embodies clean productivity with subtle, approachable styling.

## Visual Identity

### Typography
- **Headings**: Segoe UI (sans-serif) - Bold weight
- **Body**: Segoe UI (sans-serif) - Regular weight
- **Characteristic**: Friendly, readable, productivity-focused

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Notion Blue | #2383E2 | Primary accent, callouts |
| Notion Link Blue | #0075DE | Links, secondary accents |
| Near Black | #191918 | Primary text |
| Mid Gray | #6B6B6B | Secondary text, captions |
| White | #FFFFFF | Background |

### Design Principles
- Clean productivity aesthetic
- Small border-radius (5px)
- Block-based visual language
- Subtle blue accents
- Accessible and friendly

## Best Used For
- Wikis
- Documentation
- Project plans
- SOPs (Standard Operating Procedures)
- Knowledge bases
- Meeting notes
- Team handbooks

## Style Specifications
```
H1: Segoe UI 48pt, #191918 (Near Black), Bold
H2: Segoe UI 32pt, #191918, Bold
H3: Segoe UI 20pt, #191918, Bold
Body: Segoe UI 12pt, #191918
Caption: Segoe UI 10pt, #6B6B6B
```

## Document Characteristics
- Block-based table style
- Callouts and links accent usage
- Productivity-focused design
- Clean, organized layouts
- Team collaboration feel
