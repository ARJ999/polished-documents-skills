# IBM Brand Guide

## Overview
IBM is a global technology and consulting company with over a century of history. Their Carbon design system represents enterprise-grade authority with modern precision.

## Visual Identity

### Typography
- **Headings**: Arial (sans-serif) - Bold weight
- **Body**: Arial (sans-serif) - Regular weight
- **Characteristic**: Bold, confident, enterprise-grade

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| IBM Blue | #0F62FE | Primary accent, CTAs |
| IBM Black | #161616 | Primary text, headings |
| Gray 70 | #525252 | Secondary text, captions |
| White | #FFFFFF | Background |

### Design Principles
- Enterprise authority
- Sharp corners (0px border-radius)
- Grid-based layouts
- Carbon design influence
- Technical precision

## Best Used For
- Technical documentation
- Enterprise reports
- IT documentation
- System documentation
- Architecture documents
- Implementation guides

## Style Specifications
```
H1: Arial 36pt, #161616 (Black), Bold
H2: Arial 28pt, #161616, Bold
H3: Arial 18pt, #161616, Bold
Body: Arial 12pt, #161616
Caption: Arial 10pt, #525252
```

## Document Characteristics
- Grid table style
- CTA primary accent usage
- Enterprise-grade authority
- Structured, systematic layouts
- Technical credibility
