# IDBI Bank Brand Guide
## Comprehensive Visual Identity Manual

Based on official Q2 FY2025-26 Investor Presentation brand analysis.

---

## Overview

IDBI Bank (Industrial Development Bank of India) is a leading government development and commercial banking institution. The brand combines institutional trust with progressive energy through the **TOGS** principle:
- **T**eal headers
- **O**range accents
- **G**reen indicators
- **S**imple clean layouts

### Brand Tagline
- **Hindi**: प्रगति हेतु प्रतिबद्ध, सतत संरक्षण के लिए समर्पित
- **English**: COMMITTED TO PROGRESS, DEDICATED TO SUSTAINABILITY

## Color Palette

### Primary Colors

| Color Name | Hex Code | RGB | Usage |
|------------|----------|-----|-------|
| **IDBI Teal** | `#008B8B` | 0, 139, 139 | Headers, primary backgrounds, key data sections |

| Indicator | Symbol | Color | Hex |
|-----------|--------|-------|-----|
| Positive/Up | ▲ | Green | `#228B22` |

### Color Application Rule
**70-20-10 Rule**: 70% Teal/White (primary), 20% Orange (accent), 10% Secondary colors

## Typography

### Font Families

| Purpose | Font | Fallback |
|---------|------|----------|
| Headlines/Titles | Arial Bold | Helvetica Bold, Sans-serif |
| Body Text | Arial Regular | Helvetica, Sans-serif |
| Data/Numbers | Arial | Calibri, Sans-serif |

### Type Hierarchy

| Level | Size | Weight | Color | Usage |
|-------|------|--------|-------|-------|
| **H1 - Page Title** | 30pt | Bold | #008B8B (Teal) | Section dividers, cover pages |
| **H2 - Section Header** | 24pt | Bold | #333333 (Dark Grey) | Main content headers |
| **H3 - Sub-header** | 18pt | Bold | #333333 (Dark Grey) | Content sub-sections |
| **Body Text** | 12pt | Regular | #333333 (Dark Grey) | General content |
| **Table Headers** | 11pt | Bold | #FFFFFF (White) | Column/row headers on teal |
| **Table Data** | 11pt | Regular | #333333 (Dark Grey) | Data cells |
| **Caption/Notes** | 9pt | Regular | #808080 (Grey) | Disclaimers, footnotes |

## Table Styling

### Header Row
- **Background**: IDBI Teal `#008B8B`
- **Text**: White `#FFFFFF`, Bold
- **Border**: None or subtle

### Data Rows
- **Alternating**: White `#FFFFFF` and Light Grey `#F5F5F5`
- **Text**: Dark Grey `#333333`
- **Numbers**: Right-aligned
- **Text**: Left-aligned

### Highlight/Totals Row
- **Background**: Light teal tint
- **Used for**: Totals, key metrics

## Style Specifications Summary

```
H1: Arial 30pt, #008B8B (IDBI Teal), Bold
H2: Arial 24pt, #333333 (Dark Grey), Bold
H3: Arial 18pt, #333333 (Dark Grey), Bold
Body: Arial 12pt, #333333 (Dark Grey)
Caption: Arial 9pt, #808080 (Neutral Grey)

Table Headers: Teal #008B8B background, White text, Bold
Table Data: Dark Grey #333333, alternating White/Light Grey rows
Accent Line: Orange #F7941D (signature footer element)
```

---

## Brand Don'ts

- Never alter logo colors or proportions
- Never use non-brand colors for primary elements
- Never place logo on busy/cluttered backgrounds
- Never use serif fonts for body text
- Never omit the orange footer accent line (signature element)
- Never mix multiple accent colors in one section
- Never overcrowd slides with information
