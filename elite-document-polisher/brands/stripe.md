# Stripe Brand Guide

## Overview
Stripe is the leading payments infrastructure company for the internet. Their visual identity combines developer-focused clarity with sophisticated design and distinctive purple accents.

## Visual Identity

### Typography
- **Headings**: Arial (sans-serif) - Medium weight
- **Body**: Arial (sans-serif) - Regular weight
- **Characteristic**: Clean, technical, highly legible

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Stripe Navy | #0A2540 | Primary text, headings |
| Stripe Purple | #635BFF | Accents, code highlights |
| Slate Blue | #3F4B66 | Secondary text, captions |
| White | #FFFFFF | Background |

### Design Principles
- Developer-focused clarity
- Generous border-radius (16px)
- Dark navy foundation
- Purple as signature accent
- Technical precision

## Best Used For
- API documentation
- Developer guides
- Technical specifications
- Integration guides
- Product specifications
- Engineering proposals

## Style Specifications
```
H1: Arial 32pt, #0A2540 (Navy), Bold
H2: Arial 24pt, #0A2540, Bold
H3: Arial 16pt, #0A2540, Bold
Body: Arial 11pt, #0A2540
Caption: Arial 9pt, #3F4B66
```

## Document Characteristics
- Subtle shadow table style
- Code and highlights accent usage
- Developer-friendly aesthetic
- Clean, modern tech feel
- Highly readable layouts
