# Figma Brand Guide

## Overview
Figma is the collaborative design platform revolutionizing how teams create. Their visual identity bursts with creative energy through vibrant multi-color palettes.

## Visual Identity

### Typography
- **Headings**: Arial (sans-serif) - Bold weight
- **Body**: Arial (sans-serif) - Regular weight
- **Characteristic**: Clean type paired with vibrant colors

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Figma Purple | #A259FF | H1 headings, primary accent |
| Figma Blue | #1ABCFE | H2 headings |
| Figma Green | #0ACF83 | H3 headings |
| Figma Coral | #FF7262 | Secondary accent |
| Near Black | #1E1E1E | Body text |
| Mid Gray | #6B6B6B | Captions, secondary text |
| White | #FFFFFF | Background |

### Design Principles
- Design-forward creativity
- Vibrant multi-color system
- Moderate border-radius (8px)
- Color as hierarchy indicator
- Creative energy

## Best Used For
- Creative briefs
- Design documentation
- Brand guidelines
- Style guides
- Design specifications
- UI/UX documentation
- Portfolio presentations

## Style Specifications
```
H1: Arial 36pt, #A259FF (Purple), Bold
H2: Arial 26pt, #1ABCFE (Blue), Bold
H3: Arial 16pt, #0ACF83 (Green), Bold
Body: Arial 11pt, #1E1E1E
Caption: Arial 10pt, #6B6B6B
```

## Document Characteristics
- Colorful table style
- Highlights and dividers accent usage
- Creative, expressive aesthetic
- Each heading level has unique color
- Design team-oriented
