"Make this document look professional"

================================================================================

QUICK START EXAMPLES:

1. SINGLE BRAND APPLICATION
   "Hey Claude, polish my quarterly_report.docx with McKinsey style"

2. MULTI-BRAND BATCH GENERATION
   "Generate my proposal.docx in McKinsey, Deloitte, and Stripe styles"

3. ALL BRAND VARIANTS
   "Create all 10 brand versions of my document.docx"

4. BRAND RECOMMENDATION
   "Which style would work best for my financial audit report?"

BRAND RECOMMENDATIONS BY DOCUMENT TYPE:

Strategy/Board Deck        -> McKinsey

Product Documentation      -> Apple
Enterprise Technical       -> IBM

ADVANCED USAGE:

Command Line (if running directly):

  # Single brand
  python apply_brand.py report.docx mckinsey report_styled.docx

  # Multiple brands
  python apply_brand.py proposal.docx mckinsey,deloitte,stripe proposal

  # All brands
  python apply_brand.py document.docx all variants/document

  # List brands
  python apply_brand.py --list
